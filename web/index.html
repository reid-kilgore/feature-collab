Keyboard Shortcuts & Help

Shortcuts

  • ⌘+E — Add comment to selection
  • ⌘+S — Download file
  • Esc — Close modals

How to Use

  • Select text in the source editor or preview
  • Click Highlight or Comment (or use shortcuts)
  • Click an annotation card to jump to that location
  • Edit or delete annotations from the sidebar
  • Drag pane borders to resize

Terminal Workflow

  • Open from terminal: mdannotate file.md
  • Save back: Click "Copy for Terminal" and paste the command

Saving & Sharing

  • Download: Click or drag to Finder/Desktop
  • Share URL: Copies a link with the full document encoded
  • Copy for Terminal: Copies a decode command to save the file

Format: CriticMarkup

  • Highlight: {==text==}
  • Comment: {==text==}{>>note<<}
